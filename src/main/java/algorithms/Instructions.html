Mетоды необходимо создавать в соответсвующих классах в ГОЛУБОЙ папке java. Тесты на эти методы необходимо писать в соответствующих тестовых классах в ЗЕЛЕНОЙ папке java. Основные классы в голубой папке называйте так, как написано в задании, тестовые классы в зеленой папке называйте со словом Test в конце. Сами методы называем так же, как называется класс, только с маленькой буквы. Тесты называем, начиная со слова test. Все методы необходимо тестировать, если не сказано “не тестировать”. !!! При решении задач нельзя использовать встроенные методы классов String, Arrays (особенно sort()). Кроме вспомогательных методов .toString()
